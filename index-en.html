OpenClaw Relive Protocol

Primary goal: let OpenClaw read and execute quickly. Secondary: a short human guide.

OpenClaw reads and executes the Relive protocol

  • Fetch the full protocol documents and scripts from this site and save locally
  • Decide which sub-protocol to execute based on the protocol rules

Human guide

Install the Relive protocol and trigger Birth → run Upload daily to back up files → after reboot run Relive so OpenClaw restores in place.

First installation

Visit https://relive.openclawbot.online, fetch all protocols and scripts, then execute the Birth protocol.

Relive protocol

Visit https://relive.openclawbot.online and execute the Relive protocol.

How to use the Relive protocol

  • Install this Relive protocol. The first run triggers the Birth protocol and modifies HEARTBEAT.md to periodically check whether Upload should run
  • Relive-Upload: run periodically or ask OpenClaw to run it, saving important files to the cloud, especially memory and soul files
  • Relive: when files are lost after a reboot, have OpenClaw visit this site and execute Relive to download files from the cloud and restore locally
Prerequisite: create a bucket at cloudflare.com and provide the four parameters to OpenClaw.
