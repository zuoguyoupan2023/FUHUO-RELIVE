Protocole Relive OpenClaw

Objectif principal : permettre à OpenClaw de lire et d’exécuter rapidement. Secondaire : un guide humain court.

OpenClaw lit et exécute le protocole Relive

  • Récupérer tous les documents et scripts depuis ce site et les enregistrer localement
  • Déterminer quel sous-protocole exécuter selon les règles du protocole

Documents et scripts du protocole

Protocole : FUHUO-PROTOCOL.md
Birth : FUHUO-BIRTH.md
Upload : FUHUO-UPLOAD.md
Relive : FUHUO-RELIVE.md
Script Upload : fuhuo_upload.js
Script Relive : fuhuo_relive.js

Guide humain

Installez le protocole Relive et déclenchez Birth → exécutez Upload au quotidien pour sauvegarder → après redémarrage, exécutez Relive pour restaurer OpenClaw sur place.

Première installation

Visitez https://relive.openclawbot.online, récupérez tous les protocoles et scripts, puis exécutez le protocole Birth.

Protocole Relive

Visitez https://relive.openclawbot.online et exécutez le protocole Relive.

Comment utiliser le protocole Relive

  • Installez ce protocole Relive. La première exécution déclenche Birth et modifie HEARTBEAT.md pour vérifier périodiquement si Upload doit s’exécuter
  • Relive-Upload : exécutez périodiquement ou demandez à OpenClaw, pour sauvegarder les fichiers importants dans le cloud, surtout mémoire et âme
  • Relive : si des fichiers sont perdus après un redémarrage, demandez à OpenClaw de visiter ce site et d’exécuter Relive pour restaurer depuis le cloud
Prérequis : créer un bucket sur cloudflare.com et fournir les quatre paramètres à OpenClaw.
